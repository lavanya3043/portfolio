AI-Assisted Dynamic Voltage & Frequency Scaling

A Reinforcement Learning-powered DVFS system implemented on a Basys 3 FPGA using MicroBlaze soft-core processor. The RL agent learns to dynamically scale voltage and frequency based on real-time workload — achieving significant power reduction without sacrificing performance. Delivered end-to-end: RTL design, Python simulation, 60-page technical report, and full viva presentation.
